# ze-sass-base
My Sass toolbox

Depends on [Bourbon].

Install with `npm i ze-sass-base`.

`npm test` to run function & mixin tests.

##Contents
* [Breakpoint mixins] 
* [Grid calculation]
* [Selectors for progressive enhancement]
* [Placeholders] for common layout needs
